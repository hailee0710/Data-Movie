== Description ==
Make you Permalink friendly: no space, no accents, no dot, no @... and reset your Permalink if you want
Thông tin chi tiết vui lòng xem thêm tại  <a href="http://vnwebmaster.com/">Wordpress Forum</a>.

<br />
Tao permalink dep cho tieng Viet, giup bo qua cac loi hay gap trong tieng Viet bang cach chuyen toan bo link ve dang khong dau.


== Installation ==



1. Upload vietnamese_permalink's folder to the `/wp-content/plugins/vietnamese_permalink` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
